# Stack: Django + PostgreSQL + server-rendered templates with HTMX

This is a full-stack learning project; the developer knows HTML/CSS and a little Python, and no JavaScript. We chose **Django** (backend, builds on existing Python, batteries-included), **PostgreSQL** (via the Django ORM; SQLite acceptable for local dev), and **Django templates + HTMX** for the frontend.

We rejected a TypeScript / React / Next.js stack: it would require learning JavaScript, TypeScript, React, and Node all at once, risking not finishing within the 3-month budget. Templates + HTMX reuse the developer's existing HTML/CSS skills and provide interactivity without a JS framework.

A React/SPA frontend is explicitly out of scope for now; it may be revisited as a later, separate exercise.
